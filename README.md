# Block Head UGC — Reject Modernity, Return to Cube 🧊

A classic Roblox block head 3D model for UGC upload. Because Roblox deleted the original and someone had to bring it back.

## Files
- `blockhead.obj` — 3D model (8 vertices, 12 triangles, perfect cube)
- `blockhead.mtl` — Material file (classic nougat skin tone)

## How to Upload to Roblox
1. **Get UGC access** — Apply through the [Roblox Creator Hub](https://create.roblox.com/)
2. **Convert to FBX** — Open `blockhead.obj` in [Blender](https://blender.org) (free), export as FBX
3. **Upload** — Go to Creator Hub → Avatar Items → Create → Head accessory
4. **Set attachment** — Use `HatAttachment` or `FaceFrontAttachment`
5. **Submit** — Set your price and publish

## Specs
- Vertices: 8
- Triangles: 12
- Size: 1×1×1 unit (2×2×2 studs)
- Color: Nougat (#A1937F) — classic Roblox skin tone
